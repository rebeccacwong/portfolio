pandy's parlour game

pandy's parlour is a 3d web game that I designed and coded independently. Play as Pandy, a cafe owner to run your own cozy cafe. I began creating this game in November 2022, working on it intermittently as a hobby. It is still in development and I hope to add more features, including: bonuses to boost customer satisfaction, levels, ways to make modifications to the cafe and buy things with the money earned, and more.

I have created almost all the models in the game in Blender, with the exception of the foliage. I rigged the characters and some objects in the scene to enable animations, and modeled/shaded the cafe interior. I have used Unity to develop the game and code in C#. The codebase is private, please contact me if you wish to view it.
